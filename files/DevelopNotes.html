Develop|Apache|Ant|WinSetup
To use 'ant' in windows env:
	1.copy the downloaded 'apache-ant-1.9.4' folder (with 'bin', 'lib' inside)
	2.add Env Variable like: 'ANT_HOME' - 'D:\Apache\apache-ant-1.9.4'
	3.append Env Variable 'PATH' like: 'PATH' - '%PATH%;%ANT_HOME%\;%ANT_HOME%\bin\'
	4.JDK needed (not JRE)
	5.try in CMD: 'ant', should get 'Buildfile: build.xml does not exist!', 'Build failed'




Develop|Java|JDK|WinSetup
To use 'JDK' in windows env:
	1.install JDK
	2.add Env Variable like: 'JAVA_HOME' - 'C:\Program Files\Java\jdk1.8.0_40'
	3.append Env Variable 'PATH' like: 'PATH' - '%PATH%;%JAVA_HOME%\bin\;%JAVA_HOME%\lib\;%JAVA_HOME%\jre\bin\;%JAVA_HOME%\jre\lib\'
